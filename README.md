# VOPA---Voice-Operated-Personal-Assistant


## Project Description
NOVA is a Python-based Voice Operated Personal Assistant that allows users to interact with their system using voice commands.  
It can perform multiple tasks such as searching information on Wikipedia, playing YouTube videos, telling jokes, generating random facts, creating notes, and providing the current date and time.

The assistant also includes a graphical user interface (GUI) built using Tkinter for better user interaction.

---

## Features
- Voice command recognition
- Text-to-speech response
- Search information from Wikipedia
- Play YouTube videos using voice command
- Generate random facts
- Tell jokes using an API
- Create voice-based notes
- Display current date and time
- Simple GUI interface

---

## Technologies Used
- Python
- Tkinter (GUI)
- SpeechRecognition
- pyttsx3
- Selenium
- BeautifulSoup
- PyWhatKit
- Requests
- Pillow

## Project Structure
voiceassistant.py → Main program that runs the assistant<br>
wikipedia search.py → Handles Wikipedia search using Selenium
jokes.py → Fetches jokes from API
Nova.jpg → Background image for GUI
requirements.txt → Required Python libraries

## Installation

1. Clone the repository<br>
https://github.com/Ruchita-joshi/VOPA---Voice-Operated-Personal-Assistant


2. Navigate to the project folder<br>
cd VOPA---Voice-Operated-Personal-Assistant


3. Install required dependencies<br>
pip install -r requirements.txt


---

## How to Run the Project

Run the main file:
python voiceassistant.py


Click the **Speak** button and give voice commands.
